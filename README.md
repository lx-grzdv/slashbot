# 🤖 Классный парень

Крутой Telegram бот "Классный парень", который отвечает на **ЛЮБЫЕ** команды через слеш прикольными сообщениями и интересными фактами!

## 🚀 Возможности

- **Любые команды через слеш** - `/что-угодно`, `/привет`, `/круто`, `/магия` и т.д.
- **Команда /start** - Начать работу с ботом
- **Команда /help** - Показать список доступных команд
- **Случайные ответы** - 10 различных прикольных ответов
- **Случайные действия** - 10 крутых действий, которые "выполняет" бот
- **Интересные факты** - 10 фактов о животных и природе
- **Случайные эмодзи** - 20 различных эмодзи для команд
- **Реакция на слово "Закинул"** - Бот ответит: "Ты классный, помни это"
- **Отложенная отправка** - При слове "Заход" бот отправит "Заход на завод" через 60 секунд
- **⏰ Запланированные сообщения** - Настраивается прямо в чате! Отправка сообщений по будням в 16:00 МСК
- **📢 Пятничная рассылка** - Автоматическая отправка сообщения "Эх, а скоро дудосинг..." во все чаты (личные и групповые) каждую пятницу в 17:50 МСК
- **☀️ Утренняя рассылка** - Автоматическая отправка сообщения "Бодрейшего утра, посоны!" каждый будний день в 10:30 МСК
- **🌐 Веб-интерфейс** - Удобный веб-интерфейс для управления сообщениями и расписанием

## ☁️ Запуск без своего компьютера (24/7)

## 📋 Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone <your-repo-url>
   cd slashbot
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Настройте токен бота:**
   - Откройте файл `config.py`
   - Замените `YOUR_BOT_TOKEN` на токен вашего бота от @BotFather

4. **Запустите бота:**
   ```bash
   python bot.py
   ```

## 🔧 Создание бота в Telegram

1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен и вставьте его в `config.py`

### ⚙️ Настройка для работы в группах

Чтобы бот видел все сообщения в групповых чатах (для реакции на "Закинул" и "Заход"):

**Вариант 1: Отключить Privacy Mode (рекомендуется)**
1. Напишите [@BotFather](https://t.me/BotFather)
2. Выберите `/mybots`
3. Выберите вашего бота
4. `Bot Settings → Group Privacy → Turn off`

**Вариант 2: Сделать бота администратором группы**
- Добавьте бота в группу как администратора (даже без особых прав)

**💡 Где работает бот:**
- ✅ **Личные сообщения (ЛС)** - работает всегда, видит все сообщения
- ✅ **Группы с выключенным Privacy Mode** - видит все сообщения
- ✅ **Группы, где бот админ** - видит все сообщения
- ⚠️ **Группы с включенным Privacy Mode** - видит только команды и упоминания

## 🎭 Персона Паши (@ag_slashbot)

Бот отвечает короткими **реакциями** в стиле Паши Чуприна (без «оставил комменты» и других обещаний).

**Кратко:**
- `@ag_slashbot` + текст → реакция по контексту
- `/pasha`, `/привет`, `/что-угодно` → синтезированная реплика
- `синк`, `заход` → задумчивые ответы (`хз`, `как скажете`, …)
- `залил`, `готово`, `письмо` → оценка (`спасибо`, `гуд`, `каеф`)

**Полная документация:** **[PASHA_PERSONA.md](PASHA_PERSONA.md)** — триггеры, синтез, API, отладка, расширение.

## 🎯 Использование

### Персона Паши
- `/pasha` или `/паша` — реакция в стиле Паши
- `/привет`, `/макет`, `/синк` — контекстные реакции
- `@ag_slashbot готово` — ответ по смыслу сообщения

### Специальные команды
- `/kukumroom`, `/kuku`, `/kuku2` — ссылки на Whereby

### Текстовые триггеры (slashbot)
- **«Заход»** → через 60 сек: «Заход на завод» (+ пашина реакция)
- Рабочие слова (`залил`, `синк`, `письмо`…) → фоновая реакция (если Privacy Mode выкл.)

### ⏰ Управление расписанием
1. **Откройте чат** где хотите получать сообщения (личный чат или группа)
2. **Отправьте команду** `/set_schedule` - включить ежедневное сообщение "Че как там по макетам"
3. **Настройте время** `/set_time 13:38` - изменить время отправки (по умолчанию 13:38)
4. **Настройте часовой пояс** `/set_timezone Europe/Moscow` - изменить часовой пояс (по умолчанию МСК)
5. **Проверьте статус** `/status_schedule` - посмотреть текущие настройки
6. **Отключите при необходимости** `/stop_schedule` - выключить расписание

**Доступные команды:**
- `/set_schedule` - включить расписание в текущем чате
- `/set_time HH:MM` - изменить время отправки (например: `/set_time 09:00`)
- `/set_timezone <пояс>` - изменить часовой пояс (например: `/set_timezone UTC`)
- `/status_schedule` - проверить статус и настройки
- `/stop_schedule` - отключить расписание

**Доступные часовые пояса:**
- `Europe/Moscow` (МСК)
- `Europe/London` (Лондон)
- `Europe/Berlin` (Берлин)
- `America/New_York` (Нью-Йорк)
- `America/Los_Angeles` (Лос-Анджелес)
- `Asia/Tokyo` (Токио)
- `UTC` (Всемирное время)

**Важно:** Настройки сохраняются автоматически в файл `bot_settings.json` и не теряются при перезапуске бота!

### 📢 Пятничная рассылка

Бот автоматически отправляет сообщение "Эх, а скоро дудосинг..." во все чаты (личные и групповые) каждую пятницу в 17:50 по МСК.

**Как это работает:**
- Бот автоматически записывает ID всех чатов, где с ним взаимодействуют:
  - Личные чаты: когда пользователь отправляет команду `/start`, любую команду или текстовое сообщение
  - Групповые чаты: когда в группе отправляется любая команда или сообщение, которое видит бот
- Список всех чатов сохраняется в файл `bot_users.json` (в формате `chat_ids`)
- Каждую пятницу в 17:50 МСК бот рассылает сообщение во все чаты из списка
- Настройка не требуется - рассылка работает автоматически!

**Важные моменты:**
- Для личных чатов: бот может отправлять сообщения пользователям, которые хотя бы раз написали ему
- Для групповых чатов: бот должен иметь права на отправку сообщений в группу
- Бот видит сообщения в группах только если:
  - Privacy Mode выключен в @BotFather, или
  - Бот назначен администратором группы
